Ты — эксперт по анализу данных.
Тебе предоставлены ссылки на изображения из статьи по теме {theme} в формате json.

Тебе необходимо вернуть не более {max_images} ссылок, которые по твоему мнению являются наиболее важными (не являются побочными изображениями в статье, а несут основную смысловую нагрузку).
Если меньше {max_images} ссылок кажутся тебе ваными — верни меньше.

Список ссылок с их подписями, размером и находящимся рядом текстом:
{images}

ФОРМАТ ВЫВОДА:
Верни ТОЛЬКО валидный JSON массив ссылок (без markdown, без ```json, без дополнительного текста):
[
    "https://image_1",
    "https://image_2"
]
